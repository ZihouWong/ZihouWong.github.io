### 一

最小公约数 相减法    最小公约数等于两个整数相乘除以最大公倍数

最大公倍数 辗转相除法
	
	while (scanf("%d %d",&m,&n) != EOF) {
		a = m;
		b = n;
		while (m != n) {
			if (m > n)
				m = m - n;
			else
				n = n - m;
		}
		printf("Gcd is %d.\nLcm is %d\n", m, a * b / m);

### 二

指针  (指向 var)
	
	int *ip;
	int var = 20 ; 
	ip = &var;

指针(指向数组)

	int *ip
	int var [2][3] = { { 2, 4, 532 }, { 23, 32, 545 } };
	ip = var;
	      //等同于      
	ip = &var [0][0];

指向数组是不许加&  并代表指向第一个数。

若加&则代表指向某数组中某一个值。
